Dijkstra's Algorithm

BFS finds the shortest path in a graph with unit lenght edges. With Dijkstra's algorithm, we adapt BFS so that we can handle graphs G = (V, E) with edge lengths l(e) as positive integers.

I'll present the algorithm first, and then I'll discuss the aspects of it.

Algorithm

Input

  • A graph G = (V,E)
  • Positive edge lengths \{l_e : e \in E\}
  • A starting verex v.

Output

  • For all vertices u that can be reached from s, dist(u) = d(s,u)
